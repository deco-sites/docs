# Deco Docs
  
Official documentation for the [Deco](https://deco.cx) platform.

## Development

```bash
# Install dependencies
bun install

# Run in development
bun run dev
```

The server will start at `http://localhost:3000`.

## Structure

```
├── client/src/content/
│   ├── en/          # English documentation
│   └── pt-br/       # Portuguese documentation
├── server/          # Hono server
└── shared/          # Shared code
```

## Content

The documentation is organized in the following sections:

- **Getting Started** - First steps with the platform
- **Concepts** - Core concepts
- **CMS Capabilities** - CMS features
- **Developing Guide** - Development guide
- **Developing Capabilities** - Advanced development features
- **SDK** - SDK reference
- **API Reference** - API reference
- **Performance** - Performance optimization
- **Decopilot** - AI assistant
- **Self Host** - Self hosting
- **Changelog** - Change history
